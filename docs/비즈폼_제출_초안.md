## 3. 서비스 소개 — 짧은 버전 (~300자)

> 문화생활, 너무 비쌉니다. 사설 요가·필라테스는 몇 달에 몇십만 원. 그런데 지자체에는 절반 이상이 무료이고 유료도 중앙값 4.5만원인 퀄리티 높은 강좌가 15,752건이나 있습니다 — 요가부터 어학·악기·코딩·어린이 프로그램까지. 문제는 접수창이 평균 2주뿐인데 지자체는 이걸 널리 알릴 채널이 없고, 주민은 언제 열리는지 몰라 놓친다는 것(실측: 현재 접수가능 2.1%). 배움알리미는 전국 205개 기관 데이터를 정규화해 복합 검색을 제공하고, 기관별 과거 오픈 이력으로 "다음 접수가 언제 열릴지"까지 예상해 줍니다. 시민에겐 저렴한 문화생활을, 지자체에겐 홍보 창구를 — 서로 윈윈하도록 만들었습니다.

## 4. 서비스 소개 — 긴 버전 (~800자)

> **문제**: 문화생활은 비쌉니다. 사설 요가·필라테스만 해도 몇 달에 몇십만 원이 듭니다. 반면 지자체 강좌는 절반 이상이 무료(8,638/15,752건), 유료도 중앙값 4.5만원인데 내용은 체육(요가·필라테스·수영 1,400여 건)·미술공예·어학·음악·IT·자격증에 어린이 프로그램(1,400여 건)까지 온 가족 대상입니다. 그런데 이 좋은 자원이 양쪽 모두에게 막혀 있습니다 — 지자체는 205개 기관 사이트에 흩어진 강좌를 널리 알릴 채널이 없고, 주민은 평균 2주뿐인 접수창이 언제 열리는지 몰라 놓칩니다(실측: 지금 접수 가능한 강좌 2.1%). 요일×시간대×연령×무료여부를 한 번에 거르는 필터도 어디에도 없습니다.
>
> **해법**: 배움알리미는 공공데이터포털 전국평생학습강좌표준데이터(#15013110)를 정규화해 6개 도구를 제공합니다. ① 복합 필터 검색(요일·시간대·대상·무료·접수상태·정렬) ② **접수 캘린더** — 데이터에 명시된 예정 접수창과, 기관별 과거 오픈 이력 기반 "다음 오픈 예상"을 근거와 함께 제시(예: 서울 강남구청은 과거 1/6·4/28 대규모 일괄 오픈 → 다음 8/18경 예상) ③ 강좌 비교표 ④ 상세 조회 ⑤ 기관별 목록 ⑥ 필터값 안내.
>
> **LLM 웹검색으로 대체 불가한 이유**: 205개 기관 데이터의 전국 교차 집계, 질의 시점 기준 접수상태 결정론 계산, 전 기관 이력 테이블 없이는 불가능한 오픈 패턴 추정 — 모두 사전 빌드된 SQLite 위의 결정론 연산입니다.
>
> **안정성**: hot-path 외부 호출 0(사전 빌드 DB 읽기 전용), 평균 응답 ~6ms, 전 응답 24KB 이하, 모든 예측에 "예상이며 확정 아님"과 근거 병기, 데이터 출처·기준일 상시 고지. 결정론 회귀 840케이스 + 멀티턴 시나리오 QA + 실LLM 게이트로 검증했습니다.

## 5. 심사 기준별 소구 포인트 (서술형 문항 대비)

- **창의성**: "무엇을 배울까"가 아니라 **"언제 열리는가"**를 푸는 접근. 기관별 접수 오픈 이력에서 주기·월중 집중일을 추정하는 접수 캘린더는 기존 어떤 서비스에도 없는 기능. 접수창 2.1% 실측이 문제의식의 근거. **파급력**: 시민에겐 사설 대비 1/10 가격의 문화생활 접근성을, 지자체에겐 홍보 채널 없이 묻히던 공공 강좌의 활용률 제고를 — 카카오톡이라는 국민 채널 위에서 양쪽을 연결하는 윈윈 구조.
- **편의성**: 카카오톡 대화 한 번으로 "우리 동네 평일 저녁 무료 강좌" 같은 복합 조건 검색 + 비교 + 접수 알림성 정보까지. 지역 미지정 시 되묻기, 대화 맥락 필터 유지, 한국어/영어 별칭 수용 등 실LLM QA로 다듬은 대화 UX.
- **안정성**: 외부 API 의존 0(사전 빌드), 평균 6ms, 24KB 상한, 확정형 예측 표현 금지 규약, 840케이스 결정론 회귀 + 페르소나·멀티턴 QA + 원격 스모크. 데이터는 이용허락범위 제한 없는 공공데이터.
